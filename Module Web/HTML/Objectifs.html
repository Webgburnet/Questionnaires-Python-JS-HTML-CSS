Projet du DIU - EIL pour le bloc 1

Objectifs du projet

Le but de ce projet est de consolider les connaissances de la première séquence, comme la programmation Python, le versionnage collaboratif avec Git, les tests, du Web avec HTML5/CSS et JavaScript mais aussi des notions de codage de l’information numérique.

Le projet comporte :

  • 1 Module Python (GIFT, XML, JSON) : Création du questionnaire
  • 1 Module Web (HTML, CSS, JS) : Lecture du questionnaire
  • 1 Module de test (Javascript et Python)
