Exercices

Exemple de structure pour le questionnaire à choix multiple
Où se trouve le Louvre ? (100 points) A Paris A Londres A Rome

Exemple de Correction pour le questionnaire à choix multiple

Où se trouve le Louvre ? C'est Juste. La bonne réponse était A Paris. Vous avez répondu A Paris ; . 100 / 100 points.

Où se trouve le Louvre ? C'est partiellement Juste. La bonne réponse était A Paris. Vous avez répondu A Londres ; A Paris ; . 50 / 100 points.

Où se trouve le Louvre ? C'est Faux. La bonne réponse était A Paris. Vous avez répondu A Londres ; A Rome ; . 0 / 100 points.
